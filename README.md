# PagesObjectModelSelenium
В данном проекте спроектирована и частично реализована автоматизация тестирования для Формы "Пользователи" из тестового задания 1.
В качестве примера автоматизировано 3 тест кейса. Тест формы Логина и тест фильтрации формы "Пользователи" по полю "Логин".
Выделены следующие Page Objectы для дальнейшего тестирования:
* HomePage
* LoginPage
* AddUserPage
* SaveChangesPage
## Инструкция запуска
* Открыть решение с помощью Visual Studio , Меню Test-> run all tests
* .net 8.0, NUnit Version="3.14.0", "NUnit.Analyzers" Version="3.9.0", "NUnit3TestAdapter" Version="4.5.0", "Selenium.WebDriver" Version="4.38.0"
## План Дальнейшей автоматизации
Методом черного ящика продолжить проверять Позитивные\Негативные случаи согласно документации.
## План автоматизации
В первую очередь необходимо покрыть функционал редактирования, Добавления и удаления пользователей, так как это основной функционал и от него зависит 
остальной функционал работы с таблицей. Основные метрики: Процент автоматизации, Плотность дефектов, Частота отказов.
